# � Fundamentos de Python - Piedra, Papel o Tijera

Bienvenido a este proyecto de aprendizaje, diseñado para practicar los conceptos básicos de Python mediante un juego clásico y muy divertido: Piedra, Papel o Tijera.

Este repositorio combina teoría, ejercicios y una pequeña aplicación interactiva para que puedas ver cómo se aplican las estructuras fundamentales del lenguaje en un programa real.

---

## ✨ ¿Qué es este proyecto?

Este proyecto tiene como objetivo enseñar los fundamentos de Python de forma práctica y entretenida. A través del juego, se exploran ideas como:

- Variables y tipos de datos
- Operadores matemáticos y de comparación
- Condicionales
- Listas, diccionarios y estructuras de datos básicas
- Bucles y recorrido de elementos
- Entrada y salida por consola

---

## 🎮 ¿Qué hace el juego?

El programa permite jugar una partida contra la computadora. El flujo es muy simple:

1. El usuario escribe una opción: piedra, papel o tijera.
2. La computadora elige una opción al azar.
3. Se comparan ambas elecciones.
4. Se muestra si el usuario ganó, perdió o empató.
5. Si la opción ingresada no es válida, el programa lo indica.

## 🛠️ Requisitos

Para ejecutar este proyecto necesitas:

- Python 3.8 o superior
- Una terminal o consola de comandos
- Sin dependencias externas

---

## ▶️ Cómo ejecutar el juego

Abre una terminal dentro de la carpeta del proyecto y ejecuta:

```bash
python fundamentos_del_piedra_papel_tijera.py
```

Si tu sistema usa `python3`, prueba con:

```bash
python3 fundamentos_del_piedra_papel_tijera.py
```

---

## ✅ Reglas del juego

- Piedra gana a tijera
- Papel gana a piedra
- Tijera gana a papel
- Si ambas opciones son iguales, hay empate
- Si se ingresa una opción inválida, se muestra un mensaje de error
